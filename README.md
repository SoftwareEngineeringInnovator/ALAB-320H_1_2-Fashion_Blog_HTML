# ALAB 320H.1.2 - Fashion Blog HTML

This project is the plain HTML and CSS version of the Sartre's List Fashion Blog.

I recreated the website from a provided mockup using semantic HTML and regular CSS. This version was completed before converting the project into React.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Git
- GitHub
- GitHub Pages

## Project Features

- Semantic page structure using `header`, `main`, `article`, and `footer`
- Accessible header and footer navigation
- Responsive navigation using Flexbox
- Two fashion blog articles
- Accessible image descriptions
- Styled drop-cap letters
- GitHub Pages deployment

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` with Live Server.

You may also open `index.html` directly in a web browser.
